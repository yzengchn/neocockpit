# NeoCockpit 支持哪些创作素材

NeoCockpit 的创作中心目前支持五类素材：壁纸、主题素材、数字人、DIY 生图和贴纸包。不同素材的用途不一样，后续能做的事情也不一样。

<img src="https://neocockpit.cn/landing/process/04-export.webp" alt="NeoCockpit 主题包资源导出流程" />

## 五类素材怎么选

| 素材类型 | 主要用途 | 适合继续做什么 |
|:--|:--|:--|
| 壁纸 | 生成车机横屏背景 | 保存、发布灵感、作为主题背景 |
| 主题素材 | 生成主题方向、背景和图标 | 进入主题编辑器制作主题包 |
| 数字人 | 生成座舱角色形象 | 展示角色、发布灵感、做品牌角色参考 |
| DIY 生图 | 直接按提示词生成图片 | 快速试风格、补充单个视觉元素 |
| 贴纸包 | 生成车身贴纸素材 | 预览贴纸效果、下载后打印 |

如果你的目标是“车机桌面真的能使用”，优先选择主题素材。如果你的目标是“先看一个视觉方向”，可以从壁纸或 DIY 生图开始。

## 壁纸

壁纸适合生成车机横屏背景图。它可以直接作为车机桌面背景，也可以作为主题包中的背景素材。

适合描述：

- 城市夜景、自然风景、科幻座舱、越野露营等场景。
- 冷色、暖色、低饱和、玻璃质感等视觉风格。
- 横屏车机桌面、留出组件区域、不要遮挡信息等使用要求。

## 主题素材

主题素材适合生成车机主题包需要的基础资源。它通常会包含背景和图标等内容，后续可以进入主题编辑器继续制作。

适合描述：

- 想要的主题风格，例如未来科技、东方美学、运动赛车、家庭出行。
- 图标风格，例如圆角玻璃、金属描边、霓虹发光、极简线性。
- 桌面氛围，例如沉浸式、轻量、安静、强对比或高级感。

## 数字人

数字人适合生成座舱角色形象。它可以用于展示、互动想象、品牌角色设定或后续视觉探索。

适合描述：

- 年龄感、性格、服装、材质、发型和姿态。
- 座舱氛围，例如未来助手、旅行伙伴、科技管家。
- 写实、半写实、潮流、二次元或品牌吉祥物风格。

## DIY 生图

DIY 生图适合自由创作。它不会强制围绕主题包结构，可以直接按你的提示词生成图片。

适合场景：

- 快速尝试视觉方向。
- 生成单张概念图。
- 做灵感参考或社交分享。
- 补充主题制作中的某个视觉元素。

## 贴纸包

贴纸包用于生成车身贴纸素材。贴纸包不是车机桌面主题，它的目标是生成可下载、可打印、可贴到车身上的贴纸资源。

适合描述：

- 贴纸主题，例如露营、赛车、赛博、可爱、潮玩。
- 贴纸数量感，例如一组图案、多个小元素、统一套装。
- 贴纸使用位置，例如车门、车尾、后窗或整车装饰。

## 不同素材的使用边界

壁纸和主题素材更接近车机桌面主题。壁纸通常负责整体氛围，主题素材更适合继续制作完整主题包。

数字人更适合角色展示，不一定要放进车机桌面。DIY 生图更自由，适合试想法或补充素材。贴纸包面向车身外观，不是车机屏幕主题。

选择素材类型前，先想清楚你最终想要什么：

- 想要一张背景图：选壁纸。
- 想要一套车机桌面：选主题素材。
- 想要一个座舱角色：选数字人。
- 想自由生成一张图：选 DIY 生图。
- 想做车身贴纸：选贴纸包。

## 和主题包制作的关系

只有主题包才是面向车机桌面应用的完整作品。壁纸和主题素材可以作为主题包资源；数字人、DIY 图片和贴纸包也可以作为灵感或辅助素材使用，但它们不一定都会直接变成车机桌面主题。

如果你的目标是车机端应用主题，建议从“主题素材”开始，再进入主题编辑器完成主题包制作。
