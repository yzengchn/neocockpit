# 车机端如何安装和应用主题包

NeoCockpit 车机端用于浏览、预览和应用已经发布的主题包。你可以在车机上打开主题市场，选择喜欢的主题，查看预览图，然后下载并应用到车机桌面。

## 使用前准备

使用前需要确认：

- 车机已经安装 NeoCockpit 应用。
- 车机可以正常联网。
- 当前系统允许安装和使用桌面主题应用。
- 主题包已经在网页端发布到主题舱。

## 首次使用流程

首次使用可以按这个顺序完成：

1. 安装 NeoCockpit 车机应用。
2. 打开应用，进入主题市场。
3. 选择一个主题，进入详情页查看预览。
4. 点击预览，确认大图效果是否适合当前屏幕。
5. 点击应用，等待下载进度完成。
6. 回到车机桌面，检查背景、组件和 Dock 是否正常。
7. 需要调整常用应用时，在车机端设置里修改 Dock 应用。

## 安装 NeoCockpit 车机应用

将 NeoCockpit 车机端安装包安装到车机系统中。安装完成后，打开应用进入主题市场页面。

如果车机系统限制第三方应用安装，需要先在系统设置中允许安装。不同车机系统入口可能不同，通常在“设置”“安全”“应用管理”或“未知来源安装”中配置。

安装完成后，建议先确认应用可以联网。如果主题列表加载不出来，优先检查车机网络、时间设置和应用联网权限。

## 浏览主题市场

打开 NeoCockpit 车机端后，可以在主题市场中浏览已发布主题。主题列表通常会展示封面缩略图、标题和基础信息。

点击主题卡片后，可以进入主题详情页，查看更大的预览图、主题介绍和操作按钮。

选择主题时可以重点看三件事：

- 封面是否符合你的车机风格。
- 标题和描述是否说明了主题方向。
- 预览图里的组件和 Dock 是否清楚。

## 预览主题

预览用于确认主题整体效果。进入详情页后，可以查看主题的大图预览。如果有多张预览图，可以左右切换查看不同画面。

预览时重点看：

- 背景是否清晰。
- 时间、天气、导航、音乐等组件是否容易阅读。
- Dock 图标是否符合主题风格。
- 整体是否适合车机横屏使用。

## 下载并应用主题

确认主题后，点击应用按钮。车机端会下载主题资源，并显示下载进度。下载完成后，主题会应用到车机桌面。

应用后可以检查：

- 桌面背景是否更新。
- 时间、天气、导航、音乐等组件是否显示正常。
- Dock 是否展示在正确位置。
- 应用入口是否可以点击。
- 主题在当前屏幕比例下是否有明显错位。

如果下载过程中退出页面，建议重新进入主题详情页再次点击应用。下载完成前不要重复快速点击应用按钮。

## Dock 和应用入口

Dock 是常用应用入口。主题包可以提供一套默认图标风格，但车机端会优先保证真实应用可以打开。用户可以根据自己的车机应用列表调整 Dock 中的应用。

如果主题包中包含应用图标资源，车机端会尽量使用主题图标；如果没有对应图标，则使用系统原生应用图标。

## 设置页能做什么

车机端设置页用于调整使用偏好。不同版本会逐步增加更多设置，常见方向包括：

- 查看当前应用信息。
- 调整 Dock 常用应用。
- 检查主题下载和应用状态。
- 管理车机端显示偏好。

Dock 里的“应用”入口会打开车机端的应用列表。这里会展示当前设备上真实安装的应用、真实名称和真实图标，方便用户打开常用应用。

## 常见问题

### 为什么主题列表看不到刚发布的主题？

可能是主题还没有发布成功，或者车机端主题市场没有刷新。可以先回到网页端确认主题已经发布，再在车机端重新进入主题市场。

### 为什么点击应用后需要等待？

主题需要先下载到车机端，再应用到桌面。下载期间会显示进度，完成后才会切换主题。

### 为什么应用后部分图标还是原生图标？

如果主题包没有提供某个应用的图标资源，车机端会使用系统原生图标，保证应用入口仍然可用。

### 为什么主题效果和 Web 预览略有差异？

不同车机屏幕比例、字体渲染和系统环境可能会导致轻微差异。只要背景、组件位置、Dock 和应用入口整体一致，就是正常情况。

### 为什么某个应用入口打不开？

可能是当前车机没有安装对应应用，或系统限制了第三方桌面打开该应用。可以在车机端应用列表中确认应用是否存在，再调整 Dock 常用应用。
