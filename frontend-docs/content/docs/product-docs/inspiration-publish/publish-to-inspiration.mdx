# 如何发布到灵感库

灵感库用于展示已经完成并公开的作品。你在创作中心生成素材后，可以把满意的作品发布到灵感库，让其他用户浏览、点赞、评论和参考。

## 哪些作品适合发布

以下作品都可以作为灵感发布：

- 车机壁纸：适合展示整体氛围和背景视觉。
- 主题素材：适合展示一套主题方向、背景和图标风格。
- 数字人：适合展示座舱角色形象。
- DIY 图片：适合展示自由创作结果。
- 贴纸包：适合展示贴纸风格和套装效果。

## 发布入口在哪里

1. 进入创作中心。
2. 在“我的任务”中找到已完成的作品。
3. 打开作品详情页。
4. 点击“发布到灵感库”。
5. 补充标题、标签等展示信息。
6. 确认发布。

## 标题和标签怎么写

标题要让别人一眼知道这是什么。建议包含风格、场景或素材类型，例如：

- 城市夜航 HUD 主题素材
- 雪山公路车机壁纸
- 露营风车身贴纸包
- 未来座舱数字人助手

标签可以从三个角度选择：

- 风格：科技、玻璃质感、极简、潮流、东方美学。
- 场景：城市夜景、露营、越野、通勤、家庭出行。
- 类型：壁纸、主题素材、数字人、DIY 图片、贴纸包。

不要只使用“好看”“高级”“创意”这类标签，它们不利于其他用户理解作品。

## 发布前需要确认什么

发布前建议先检查这些内容：

- 作品已经生成完成，不是排队中或生成失败状态。
- 封面图可以正常展示。
- 标题能表达作品主题，不要只保留临时名称。
- 标签能描述风格、场景或素材类型。
- 图片没有明显错误，例如主体缺失、内容模糊、比例不合适。
- 如果是主题素材，背景和图标风格应尽量统一。

## 哪些内容不适合发布

以下内容建议先重新生成或整理后再发布：

- 图片明显模糊、拉伸、比例不适合展示。
- 标题为空或无法说明作品内容。
- 主题素材里背景和图标风格完全不一致。
- 贴纸包只有零散图案，无法看出套装风格。
- 作品包含不适合公开展示的内容。

## 发布后会发生什么

发布后，作品会出现在公开灵感库列表中。其他用户可以进入详情页查看作品预览、点赞和评论。灵感库更适合展示 AI 生成结果和创意方向，不等同于已经完成的主题包。

你仍然可以在自己的创作中心查看原任务。发布到灵感库不会自动把作品变成主题包，也不会自动应用到车机端。

## 发布后如何管理

发布后，你可以继续在自己的创作中心找到原作品。对于已经发布的作品，可以根据页面入口查看详情、继续制作主题包，或在不想公开展示时取消发布。

如果你后续基于这份素材制作了主题包，灵感库作品和主题包仍然是两件不同的作品。灵感库展示创作结果，主题舱展示可以在车机端应用的主题。

## 和制作主题包的关系

发布灵感和制作主题包是两个不同动作：

- 发布灵感：让别人看到你的 AI 创作结果。
- 制作主题包：把素材、布局和组件做成可在车机端使用的主题。

如果你发布的是主题素材，后续还可以继续进入主题编辑器，把背景、图标和组件组合成主题包。

## 如何让作品更容易被看懂

- 标题写清主题，例如“城市夜航 HUD 主题素材”。
- 标签选择风格和场景，例如科技、夜景、越野、可爱、玻璃质感。
- 描述里说明适合用在哪里，例如车机壁纸、主题包素材、贴纸打印。
- 不要把多个完全不同风格的作品混在一起发布。

## 什么时候应该制作主题包

如果你的目标只是展示 AI 生成效果，可以发布到灵感库。如果你的目标是让车机端使用这套视觉，就应该继续制作主题包，并在主题编辑器里完成背景、组件、Dock 和预览图。

## 发布检查清单

- 标题清楚。
- 标签能说明风格和场景。
- 封面图正常。
- 详情页里的主要图片能代表作品。
- 如果是主题素材，后续有继续制作主题包的价值。
- 如果是贴纸包，能看出统一贴纸风格。
