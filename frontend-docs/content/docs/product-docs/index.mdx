# 产品文档

NeoCockpit 是面向智能座舱的 AI 创作平台。你可以用它生成车机壁纸、主题素材、数字人形象、DIY 图片和贴纸包，也可以把素材制作成专属车机主题包，发布后在车机端安装使用。

## 你可以在这里做什么

- 生成适合车机屏幕的高清壁纸。
- 生成主题素材包，用于后续制作车机桌面主题。
- 生成数字人形象资源，用于座舱展示和互动体验。
- 使用 DIY 生图直接创建自定义图片。
- 生成贴纸包，预览贴纸效果后下载用于打印。
- 把作品发布到灵感库，获得其他用户的浏览、点赞和评论。
- 使用主题编辑器制作车机主题包，并发布到主题舱。
- 在车机端安装 NeoCockpit 应用，浏览、预览和应用主题。

## 新用户从这里开始

1. 注册或登录 NeoCockpit。
2. 进入创作中心，选择壁纸、主题素材、数字人、DIY 生图或贴纸包。
3. 输入你的创作描述，等待 AI 生成素材。
4. 生成完成后，可以把作品发布到灵感库，也可以继续进入主题编辑器。
5. 在主题编辑器中摆放时间、天气、导航、音乐、Dock 等车机桌面组件。
6. 保存并发布主题包后，在车机端 NeoCockpit 应用中预览和应用。

## 按你的目标选择路线

如果你只是想快速体验 AI 创作，可以从“壁纸”或“DIY 生图”开始。它们步骤最短，适合快速看到风格结果。

如果你想做一套完整车机桌面，建议从“主题素材”开始。主题素材更适合进入主题编辑器，继续设置背景、组件、Dock 和预览图。

如果你想把作品分享给其他用户，可以先发布到灵感库。灵感库展示的是 AI 创作结果，适合获得浏览、点赞和评论。

如果你想在车机上真正使用主题，需要把素材制作成主题包，并发布到主题舱。车机端应用会从主题舱浏览、预览和应用主题。

## 创作素材有哪些类型

- 壁纸：生成适合车机横屏展示的高清背景图。
- 主题素材：生成背景、图标等用于制作车机主题的素材。
- 数字人：生成座舱数字人形象和展示资源。
- DIY 生图：直接按你的提示词生成图片。
- 贴纸包：生成车身贴纸素材，可下载后用于打印。

## 一次完整创作会经历什么

完整流程通常是：

1. 在创作中心生成素材。
2. 在作品详情页检查生成结果。
3. 满意的作品发布到灵感库，让其他用户看到。
4. 适合做主题的素材进入主题编辑器。
5. 在主题编辑器中完成布局、组件、Dock 和整体风格。
6. 发布主题包到主题舱。
7. 在车机端应用中查看预览，并应用到车机桌面。

你也可以只完成其中一部分。例如，只生成壁纸用于保存，或只发布灵感用于展示，不一定每次都制作主题包。

## 如何发布灵感

生成完成的作品可以发布到灵感库。发布后，其他用户可以在灵感库中浏览作品、查看详情、点赞和评论。你也可以通过灵感库查看其他人的创作，寻找可继续制作主题包的视觉方向。

## 如何制作主题包

主题包制作是 NeoCockpit 的重点能力。你可以把已生成的主题素材带入主题编辑器，在画布中设置背景、组件位置、Dock 应用入口和整体视觉风格。完成后保存草稿、生成预览，并发布到主题舱，让车机端可以浏览和应用。

## 车机端如何使用

在车机端安装 NeoCockpit 应用后，可以进入主题市场浏览已发布主题，查看主题详情和预览图。选择喜欢的主题后，下载并应用到车机桌面。
