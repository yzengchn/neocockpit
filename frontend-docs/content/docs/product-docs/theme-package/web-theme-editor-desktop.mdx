# 主题编辑器如何制作车机桌面

NeoCockpit 的主题编辑器用于把 AI 生成素材制作成完整车机主题包。你可以设置背景，摆放时间、天气、导航、音乐和 Dock 等组件，调整整体视觉效果，然后保存、预览并发布。

## 什么时候需要进入主题编辑器

当你不只是想展示一张图，而是想做一个车机桌面主题时，就需要进入主题编辑器。主题编辑器会把素材、组件和布局组织成主题包，让车机端可以预览和应用。

适合进入主题编辑器的情况：

- 你已经生成了主题素材，想做成完整桌面。
- 你想把壁纸、图标和 Dock 组合起来。
- 你想调整时间、天气、导航、音乐等组件的位置。
- 你想发布到主题舱，并在车机端使用。

## 制作前准备

建议先准备这些内容：

- 一张适合横屏车机桌面的背景图。
- 一组风格统一的图标资源。
- 一个清楚的主题名称。
- 一段简短描述，说明主题风格和适合场景。
- 想要展示的组件，例如时间、天气、导航、音乐和 Dock。

## 一个主题包通常包含什么

一个完整主题包不只是背景图。它通常包含：

- 背景：决定整体氛围，也影响文字和组件的可读性。
- 组件：时间、天气、导航、音乐、Dock 等车机桌面内容。
- 图标：Dock 和应用入口使用的视觉资源。
- 布局：每个组件在车机屏幕上的位置和大小。
- 预览图：给用户在主题舱和车机端判断效果。
- 标题和说明：告诉用户主题风格、适合场景和主要特点。

制作时要把这些内容放在一起看。只做一张好看的背景图，不等于完成了车机桌面主题。

## 基础制作流程

1. 新建主题包，或从已生成的主题素材进入编辑器。
2. 设置主题标题、描述和封面信息。
3. 选择背景图，并调整整体视觉效果。
4. 添加桌面组件，例如时间、天气、导航和音乐。
5. 添加 Dock，设置展示位置和应用入口数量。
6. 调整组件大小、位置和层级，让桌面信息保持清晰。
7. 生成预览图，检查和实际车机桌面是否接近。
8. 保存草稿。
9. 确认无误后发布到主题舱。

## 推荐制作顺序

建议按这个顺序做，返工会少一些：

1. 先确定背景图和整体色调。
2. 再放时间组件，因为时间通常是最重要的信息。
3. 然后放天气、导航、音乐等信息卡片。
4. 最后放 Dock，确定常用应用入口和排列方向。
5. 统一检查组件边距、字号、透明度和阴影。
6. 生成预览图，对比画布和预览是否一致。
7. 保存草稿，再确认发布。

如果一开始就堆很多组件，后面很容易出现文字遮挡、Dock 太挤、背景抢信息的问题。

## 组件怎么摆放更合理

- 时间组件适合放在左上、右上或视觉焦点附近。
- 天气组件适合放在侧边或中部信息区。
- 导航组件适合放在驾驶者容易扫视的位置。
- 音乐组件适合和导航、天气组成信息卡片区。
- Dock 适合放在底部、顶部或左右边缘，作为常用应用入口。
- 背景图不要和文字、图标抢视觉焦点。

## 组件设置建议

时间组件建议保持清晰，不要被背景高光压住。时间、星期、日期要能快速扫读。

天气组件建议放在不干扰驾驶信息的位置。城市名称、温度和天气状态不要太小。

导航组件建议留足宽度，避免路线信息被压缩。它更适合放在左侧或中间偏左区域。

音乐组件可以和导航、天气组成信息区，但不要过度装饰。车机桌面首先要能读懂。

空调、车辆状态等组件适合作为辅助信息，不建议放在视觉中心。

## Dock 应用入口

Dock 是车机桌面上的常用应用入口。编辑器中可以设置 Dock 的位置、排列方式和应用槽位。车机端实际使用时，用户也可以根据设备上的真实应用进行调整。

默认建议从主题、天气、导航、音乐、应用、空调等常用入口开始，保持数量适中，避免底部过挤。

Dock 可以放在底部、顶部、左侧或右侧。底部更接近常见车机桌面，左侧和右侧适合更强的座舱感设计。

设置 Dock 时建议检查：

- 图标数量是否适合当前宽度或高度。
- 图标之间的间距是否均匀。
- 横向排列和纵向排列是否都没有挤压。
- 应用入口是否符合用户常用习惯。
- 空调等固定功能是否和普通应用入口保持统一视觉。

如果你想做更简洁的桌面，可以减少 Dock 项目数量，让背景和核心组件更突出。

## 保存、预览和发布

保存草稿用于保留当前编辑进度。生成预览用于检查主题整体效果。发布到主题舱后，车机端用户才能在主题市场中看到并应用这个主题。

发布前建议检查：

- 标题和封面是否清楚。
- 背景、组件和 Dock 是否对齐。
- 文字是否被背景遮挡。
- 图标是否容易识别。
- 预览图是否能代表主题实际效果。

## 发布前检查清单

发布前建议完整检查一次：

- 背景图清晰，和主题名称一致。
- 时间、天气、导航、音乐等组件没有互相遮挡。
- 所有组件都在车机屏幕可见范围内。
- Dock 位置合理，图标没有贴边或重叠。
- 文字在深色和浅色背景上都能看清。
- 预览图和编辑器画布效果基本一致。
- 标题、描述和标签能说明主题特点。
- 保存草稿成功后，再确认发布。

## 常见问题

### 为什么预览图和编辑器画布看起来不完全一样？

不同屏幕比例和显示环境会带来轻微差异。重点检查组件位置、文字可读性、Dock 排列和整体氛围是否一致。

### 为什么组件放上去以后背景变得不好看？

车机桌面不是纯图片展示。背景需要给组件留出可读区域。如果背景太花，可以降低对比度，或把主要组件放到更干净的位置。

### 为什么 Dock 图标看起来太挤？

通常是图标数量、排列方向和间距不匹配。可以减少入口数量，或把 Dock 从底部改到侧边。

### 什么时候可以发布？

当主题在编辑器画布和预览图中都能清楚展示，并且车机端用户能理解它的风格和用途时，就可以发布到主题舱。
