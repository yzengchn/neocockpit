# Android 文档

Android 文档用于说明 NeoCockpit 车机端如何消费 Web 主题编辑器发布的主题包。

## 核心流程

1. 从主题市场接口获取已发布主题包列表。
2. 进入主题详情页，展示封面、预览图、组件能力和资源信息。
3. 下载主题包并读取 manifest。
4. 校验主题包资源、组件布局和版本信息。
5. 将主题应用到 Android Launcher 桌面。

## 当前主题能力

- 背景资源：支持主题背景展示。
- 时间组件：展示车机桌面时间信息。
- 导航组件：作为地图入口和导航信息展示组件。
- 音乐组件：作为媒体应用入口和基础播放状态展示组件。
- 天气组件：展示天气信息入口和天气卡片。
- Dock 组件：承载常用应用入口和资源化图标。

## 后续补充

后续可以继续补充 manifest 字段、资源目录协议、组件映射矩阵、昼夜模式和 SystemUI 资源包协议。
